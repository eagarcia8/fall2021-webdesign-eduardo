Text can be modified in many ways. It can be italized using font-style, it can be bolded using font-weight, it can be underlined, and crossed out using text-decoration.

The line-height property can be used to increase or decrease the space between lines. NOTE that line-height includes the ENTIRE height, not just space between. The following paragraph has a (total) line height of 28 pixels.

Spacing

Margin and Padding can be applied to any block level and inline-block level element.

This container has margin, this means it pushes its siblings away. Margin is not considered part of the container size. This container has 40 pixels of margin on the top, left, right, and bottom.

This container doesn't have margin, but it does have padding, meaning it pushes its children "inward". Padding is considered part of the container size. This container has 40 pixels of padding inside of itself on the top, left, right, and bottom.
