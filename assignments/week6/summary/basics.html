Text Content

The Paragraph element (p) is used for normal text.

The Heading element (h1-h6) is used for content titles

The blockquote (blockquote) is used for text that adds to the main topic but isn't part of the main topic.

Lists

  • The unordered list (ul) is for a list of text (usually) without an order.
  • The unordered list always has list items (li) as children.
  • The unordered list by default uses bullet points.
  1. The ordered list (ol) is for a list of text (usually) with an order.
  2. The ordered list always has list items (li) as children.
  3. The unordered list by default uses numbers.
dl
The dictionary list (dl), is a list for terms (usually).
dt
The dictionary list has only two type of children, dictionary term (dt) and dictionary description (dd)
dd
The dictonary description by default is indented in relation to the term.

Tables

A table should be for tabular data. The common children for table are thead, tbody, tr, td, and th. By default tables do not have ANY styling.

table is the container for the table contents.
tr is for creating one row in the table.
td is for the cells of that row, they go inside of tr.
th is for the column or row "name" or "title"
thead is for a special row designated specifically for 'th's
tbody is for the main tabular rows.

Forms

Form inputs usually are LINKED to a label, each input must have a label attached except for the types button, reset, submit. They are linked using the attributes id and for.






Images

The img element is used for displaying images on a website. img elements REQUIRE the alt attribute, and can optionally have the title attribute (not the same as title element). Images are by default inline-block elements.
