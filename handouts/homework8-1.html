Homework8-1

Please make sure to fix previous homework assignments that this homework depends on. If these previous homework assignments are not completed, this homework cannot pass.

This homework follows the homework assignment 6-1. Make sure you have Homework 6-1 documents ready as the grade for the entire project is based on how well the planning reflects the actual website, that is to say if the planning looks different compared to the actual website the final project grade can be reduced or possibly fail Homework 8-1.
Expect to dedicate about two to four hours to complete this homework!

  • Create a folder called portfolio-website inside of the projects folder. You will be placing all content from this homework assignment into this folder.

  • Create the first HTML file which will be the base for your web pages. This HTML file should be called base.html. Create a CSS file and call it styles.css. Link this CSS file to your base.html file. Focus on the design and structure of the Header and Footer HTML elements in your base.html file. Create your Navigation element as well. Do not worry about the Main HTML element just yet. Make sure to git add and commit after this step.

  • Once you are completely satisfied with the base.html design, make a copy of this HTML file for each of your web pages. Rename each of these copies to their proper web page name. Remember that whatever your home page will be, it must be called index.html. Once you have an HTML file for each of your web pages, fill in the Main HTML element for each of them. In the Main section of each web page, use IDs and Classes to avoid CSS conflicts with the other web pages. Git add and commit after this step.

  • Format your text and images as described in the Homework 6-1 assignment. Git add and commit after this step.

  • Test all your links, test your HTML code with the W3 Validator, and test your CSS code in the W3 CSS Validator. Fix any issues you find with the time you have left before submitting this homework assignment.

  • Git add and commit all files. The message for this commit should be Portfolio Version 1.0. Push to your GitHub account.
