Homework5-1

For this homework, you will installing Krita and Inkscape software. This software is free and open-source and a great start to editing and creating images.

Downloading & Installing Krita

Go to the Krita Download page and click on the blue installer button.

  • For Windows, you should have downloaded an EXE file. Double click this file and follow the instructions in the wizard, make sure to leave all options to their defaults.
  • For Macs, you should have downloaded a DMG file. Double click this file and it should open a new window. Drag the Krita icon to the Applications Folder.

Downloading & Installing Inkscape

Go to the Inkscape Download page and click on your Computer 's Operating System (below the red image).

  • For Windows, click on either 64-bit or 32-bit button below the red image. After clicking on the 64-bit or 32-bit button, click on the EXE button to download an EXE file. Double click this file when it’s done to start the installation wizard. Leave all options to the default value as you go through the wizard.

    • If you don’t know your CPU architecture (32-bit vs 64-bit) please check out the official Inkscape documentation.

  • For Macs, it should have started a DMG download automatically. When the download completes, open the DMG file and drag the Inkscape icon to the Applications folder.
