Homework6-1

Please make sure to fix previous homework assignments that this homework depends on. If these previous homework assignments are not completed, this homework cannot pass.

The portfolio website will be the second project that you will work on and complete by the end of the next week. Creating a website may seem easy at first but there are many parts to the design of a website that planning becomes necessary. For this homework assignment, you will be planning your portfolio website by giving yourself design and content goals and notes. This includes colors, images, font sizes, the amount of pages for your website, what each web page will be about, and anything else you believe you would like to make a note of for your portfolio project. Expect to dedicate about one to two hours to complete this homework!

Note there should be no CSS and HTML code submitted for this homework assignment.

Create a folder called portfolio_planning inside of your projects folder. Places all content created for this homework into this folder.

  • Create a text file and call it “design_brief.txt”. In this file describe in sentences your choices of colors you would like to use for your website. You are free to add reasons for your choice so you can track why you chose them. These color choices must at a minimum cover body, header, and link text as well as background color for each of your elements.

    • Try picking out fonts from Google Fonts.
    • If you believe you can make use of icons, try gathering a list of icons from FontAwesome.
    • Use a color wheel to pick out colors, themes, and color codes.

  • Gather the images you would like to have on your web page, at a minimum you should have a portrait of yourself. Some images you can have are icons for tech skills you already have like Microsoft Word or Adobe Photoshop. Other images can be related to what you do. For example if you design business cards, you can have images of business cards you have designed, or generic business cards. Keep a note of image licensing if you do not own the image!

  • In the design_brief.txt file you created in the first step, write out each web page of your portfolio in capital letters, and under each web page write the text content you would like to have on that web page. At minimum you should have text for the following:

    • The about me page (home page)
    • The contact page
    • The projects page

  • There should be a minimum of two paragraphs in the about me page. There is no minimum paragraphs for the contact or projects page, but should still include text.

  • Makre sure to locate and prepare your resume as a PDF file as it will be added to the final website code.
